# Klasifikasi Golongan Penyakit Kanker Payudara Menggunakan Model Regresi Logistik
Proyek ini memodelkan diagnosis kanker payudara pada dataset Breast Cancer Wisconsin (Diagnostic) menggunakan regresi logistik untuk mengklasifikasikan kasus sebagai jinak atau ganas. Dua model dievaluasi, yaitu sebelum dan sesudah penerapan PCA, dan hasil menunjukkan bahwa model dengan PCA memberikan performa yang lebih baik dengan akurasi 93.29%, sensitivity 88.30%, dan specificity 100%. Evaluasi melalui kurva ROC juga menunjukkan kualitas prediksi yang sangat tinggi, mendekati 99.5%, menandakan model hampir tidak melakukan kesalahan dalam klasifikasi. Dengan demikian, regresi logistik berbasis PCA menjadi pendekatan yang lebih efektif, meskipun peningkatan lebih lanjut tetap diperlukan untuk meminimalkan kesalahan prediksi.
